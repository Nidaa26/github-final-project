# Simple Interest Calculator

A simple shell script to calculate simple interest given principal, annual rate of interest, and time period in years.

## Introduction

This project provides a simple bash script that computes **Simple Interest** using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Inputs

- **Principal** – The initial amount of money
- **Rate** – Annual rate of interest (in %)
- **Time** – Time period (in years)

## Output

- **Simple Interest** – The calculated interest amount

## Usage

```bash
bash simple-interest.sh
```
